## 📊 文档分类说明

### Changelog (版本日志)
- **用途**: 记录每个版本的新功能、改进和 Bug 修复
- **对象**: 用户和开发者
- **推荐阅读**: 了解最新版本有什么新功能

### Guides (使用指南)
- **用途**: 帮助用户安装、配置和使用插件
- **对象**: 新用户、需要配置帮助的用户
- **推荐阅读**: 第一次使用时查阅

### Tests (测试文档)
- **用途**: 验证各个功能是否正常工作
- **对象**: 开发者、QA、高级用户
- **推荐阅读**: 升级后进行功能验证

### Summaries (总结文档)
- **用途**: 记录开发历程、功能升级、优化改进
- **对象**: 开发者、项目管理
- **推荐阅读**: 了解项目演进过程

### Releases (发布说明)
- **用途**: 发布前的演示说明和发布公告
- **对象**: 所有用户
- **推荐阅读**: 新版本发布时
