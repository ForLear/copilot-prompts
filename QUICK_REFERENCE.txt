  cd /path/to/copilot-prompts
  open agent-manager.html

📋 推荐配置

  Vue 3 前端项目:
    ✅ Vue 3 Agent
    ✅ TypeScript Agent
    ✅ i18n Agent

  VitaSage 工业项目:
    ✅ VitaSage Agent
    ✅ TypeScript Agent
    ✅ i18n Agent

  全栈项目:
    ✅ 所有 4 个 Agents
    ✅ 相关 Prompts

🔧 应用流程

  1. 打开管理界面
     open agent-manager.html

  2. 选择 Prompts
     勾选需要的 Agent/Prompt 卡片

  3. 生成配置
     点击 "✅ 生成配置"

  4. 应用到项目
     cd your-project
     mv ~/Downloads/apply-prompts.sh ./
     chmod +x apply-prompts.sh
     ./apply-prompts.sh

  5. 重新加载 VS Code
     Cmd+Shift+P → "Reload Window"

⚡ 快捷操作

  全选:   点击 "☑️ 全选"
  清空:   点击 "☐ 清空"
  搜索:   输入关键词实时过滤
  分类:   侧边栏点击分类
  帮助:   点击 "❓ 使用帮助"

📊 界面功能

  统计卡片:
    - 总计: 所有可用 Prompt
    - 已选择: 当前选中数量
    - Agents: Agent 类型
    - Prompts: Prompt 类型

  侧边栏:
    - 全部: 显示所有
    - Agents: 仅 Agent
    - Prompts: 仅 Prompt
    - 行业专用/Vue/通用

  卡片信息:
    - 标题和路径
    - Agent/Prompt 标签
    - 描述和技术栈标签
    - 复选框选择

💡 使用技巧

  搜索示例:
    "typescript" → TypeScript 相关
    "vue3"       → Vue 3 相关
    "i18n"       → 国际化相关

  批量操作:
    全选 → 取消部分 → 生成配置

  多项目:
    为不同项目生成不同配置脚本
