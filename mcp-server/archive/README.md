# Archive 归档目录

此目录用于存放历史文档和测试文件，保持项目根目录清洁。

## 目录结构

```
archive/
├── docs/          # 历史文档
│   ├── CLAUDE_SETUP.md
│   ├── DEVELOPMENT_ROADMAP.md
│   ├── GETTING_STARTED.md
│   ├── MIGRATION_GUIDE.md
│   ├── TROUBLESHOOTING.md
│   └── USAGE_GUIDE.md
│
└── tests/         # 测试文件
    ├── test-*.cjs
    └── test-*.sh
```

## 说明

- **docs/** - 存放历史版本的文档、指南、说明等
- **tests/** - 存放开发过程中的测试脚本和调试文件

## 访问历史文档

如需查看历史文档，请访问对应子目录：
- 安装指南：`docs/GETTING_STARTED.md`
- 使用手册：`docs/USAGE_GUIDE.md`
- 故障排查：`docs/TROUBLESHOOTING.md`
- 等等...

---

**注意**：主要文档请查看项目根目录的 [README.md](../README.md) 和 [CHANGELOG.md](../CHANGELOG.md)
